# voice-calling

This is a real-time Video + Voice Calling Web Application built using React.js, Tailwind CSS, and ZegoCloud.
Users can join a room using a unique room ID and instantly start high-quality video or voice calls.
The app is lightweight, responsive, and works smoothly on desktop as well as mobile devices.
